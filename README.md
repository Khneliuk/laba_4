Завдання 1

	Створити додаток, який задовольняє вимогам, наведеним в завданні. Наслідування застосовувати тільки в тих завданнях, в яких воно логічно обґрунтоване. Аргументувати належність класу кожного створюваного методу і коректно перевизначити для кожного класу методи Equals, GetHashCode, ToString. При виклику будь-якого методу класу, виводити на екран текстове повідомлення.
Створити об'єкт класу Держава, використовуючи класи Область, Район, Місто. Методи: вивести на консоль столицю, кількість областей, площу, обласні центри.

Завдання 2

	Сервіс таксі. Сервіс таксі зберігає дані про поїздки: стандартні та міжміські. Кожна поїздка має унікальний код, відстань (км) та базову ціну. Система повинна знати загальну кількість створених поїздок. Створіть базовий клас Trip. Код, відстань і ціна зберігайте у закритих полях (private), код зробіть незмінним (readonly), а для підрахунку використайте спільне поле (static). Додайте віртуальний метод (virtual) GetInfo() та метод, що визначає, чи поїздка “довга” (наприклад, більше км). Створіть похідні класи CityTrip та IntercityTrip: у CityTrip додайте поле район, у IntercityTrip — назву міста призначення, і перевизначте GetInfo() (override). У Main() створіть кілька поїздок, додайте їх у список базового типу Trip, виведіть інформацію, підрахуйте загальну суму цін та кількість “довгих” поїздок.
